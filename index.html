Yii2-betssonsports

Yii2 Betsson Sports API connector

View on GitHub Download .zip Download .tar.gz

Design and Purpose

The Yii2-betssonsports extension allows you to download and locally store a Betsson sportsbook feed. It comes with Yii2 models which allow you navigate and access the dataset.
